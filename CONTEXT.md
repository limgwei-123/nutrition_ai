# Context Glossary

## Terms

### Prediction Request

A user's text description of food or a meal submitted for calorie estimation.

### Prediction

A single best calorie estimate produced from a Prediction Request, including the interpreted food, confidence, and calories.

### Feedback

A user's judgement of whether a Prediction was correct, with optional corrected food and calorie values.

### Metrics

Summaries calculated from stored Predictions and Feedback to describe prediction volume, quality, and performance.

### Food Knowledge

Static food and calorie information used to support a Prediction before any external nutrition API exists.

### Anonymous User

A person using the app without an account or profile.
